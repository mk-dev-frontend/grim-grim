# grim-grim

Grim & Grim landing page (technical task)

## Live Demo

https://mk-dev-frontend.github.io/grim-grim/

## Purpose

Demonstrates frontend skills with interactive components, adaptive layout, and modern libraries.  
_Created to showcase frontend abilities and practical implementation of interactive components._

## Key Features

- **Splide Slider** — looped slider for members with responsive breakpoints and custom navigation.
- **GLightbox** — image popup for gallery items with smooth animation.
- **Ticket Popup** — interactive ticket selection: calculates price based on quantity and displays event info dynamically.

## Libraries Used

- [Splide.js](https://splidejs.com) — for the slider
- [GLightbox](https://glightbox.github.io/) — for image popups

## Notes

- Adaptive layout for multiple screen sizes.
- Modern JavaScript (ES6+) for interactive functionality.
- Integrated third-party JavaScript libraries for interactive UI components.
